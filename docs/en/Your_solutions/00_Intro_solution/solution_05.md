# Physics Problem: Trigonometry

## 1. Key Definitions and Formulas

Before solving the problem, we need to understand the variables and trigonometric relations:

*   **Magnitude ($|\vec{A}|$):** The length of the vector, which is given as 15.
*   **Angle ($\theta$):** The angle the vector makes with the horizontal axis, given as $60^\circ$.
*   **Horizontal Component ($A_x$):** The projection of the vector on the x-axis.
    $$A_x = |\vec{A}| \cos(\theta)$$
*   **Vertical Component ($A_y$):** The projection of the vector on the y-axis.
    $$A_y = |\vec{A}| \sin(\theta)$$

## 2. Solving the Problem

**Task:** Calculate the horizontal and vertical components of vector $\vec{A}$.

### Step-by-Step Derivation:

1.  **Identify the given values:**
    $$|\vec{A}| = 15$$
    $$\theta = 60^\circ$$

2.  **Calculate the horizontal component ($A_x$):**
    Use the exact value $\cos(60^\circ) = \frac{1}{2}$.
    $$A_x = 15 \cdot \cos(60^\circ)$$
    $$A_x = 15 \cdot \left(\frac{1}{2}\right) = 7.5$$

3.  **Calculate the vertical component ($A_y$):**
    Use the exact value $\sin(60^\circ) = \frac{\sqrt{3}}{2}$.
    $$A_y = 15 \cdot \sin(60^\circ)$$
    $$A_y = 15 \cdot \left(\frac{\sqrt{3}}{2}\right) = \frac{15\sqrt{3}}{2} \approx 12.99$$

### Final Answer:
The components of the vector are:
$$A_x = 7.5$$
$$A_y = \frac{15\sqrt{3}}{2} \approx 12.99$$
